#ENG
GET "/api/books/" Get the list of books
GET "/api/books/{book_id}" Get one book depending on his id (book_id)
POST "/api/books" Add a book to database
PUT "/api/books/{book_id}" Update book data depending on his id (book_id)
DELETE "/api/books/{book_id}" Delete book depending on his id (book_id)

#FR
GET "/api/books/" Obtenir la liste des livres
GET "/api/books/{book_id}" Obtenir un livre à partir de son id (book_id)
POST "/api/books" Ajouter un livre à la base de données
PUT "/api/books/{book_id}" Mettre à jour un livre en fonction de son id (book_id)
DELETE "/api/books/{book_id}" Supprimer un livre avec son id (book_id)

#DEPENDENCES

fastapi
uvicorn
odoorpc
pydantic

#RUN THE APPLICATION
python odoo-bin -c odoo.conf

#OTHER INSTRUCTIONS
#ENG
install postgresql et make sure to create a database according to the attribute "db_name" in odoo.conf file
So make sure you have user in postgres database with names & password according to the attribute "db_user" & "db_password" in odoo.conf file
execute this command in your bash : "pip install -r requirements.txt"

#FR
installez postgresql et assurez vous de créer une base de données en fonction de l'attribut "db_name" dans le fichier odoo.conf
Aussi assurez vous que vous avez un utilisateur dans la base de données dont le nom et le mot de passe seront "db_user" et "db_password" dans le fichier odoo.conf
exécuter cette commande dans votre terminale : "pip install -r requirements.txt"

thanks for all
